# 图书管理数据库项目

## 项目简介

这是一个简单的图书管理数据库项目——我们需要实现一个集前后端于一体的 `C++` 程序，让它能够支持对图书借阅事务的记录与处理。

图书馆的借阅系统需要支持如下几种功能：

- 记录所有的读者相关信息，包括读者的姓名，唯一身份 ID （学号）, 年龄等相关的基本信息。
- 记录所有的图书借取记录，并支持对图书借取记录的查询。
- 支持新建读者信息，插入借阅记录。
- 支持对借阅记录进行查询。
